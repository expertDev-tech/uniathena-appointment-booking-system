# UniAthena Appointment Booking System

A RESTful Appointment Booking System developed as part of the UniAthena Laravel Technical Assessment.

## Technology Stack

- PHP 8.2+
- Laravel 12
- MySQL
- REST API
- Laravel Queue (Database Driver)

## Features

- Doctor Availability Management
- Dynamic Availability Slot Generation
- Appointment Booking
- Appointment Cancellation
- Appointment Rescheduling
- Asynchronous Email Notifications
